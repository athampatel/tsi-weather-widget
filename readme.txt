== Description ==

 Weather Widget - Using open weather map API

Major features in Akismet include:

* Ability Set Diffrent Location.
* Display Widget on Admin Dashborad.

== Installation ==

Upload the TSI Weather Widget plugin to your blog, activate it, and then enter your openweathermap.org API key.

1, 2, 3: You're done!
